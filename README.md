# 🏢 Smart Procurement Vendor Management System

This project is a **vendor and procurement management system** designed to streamline organizational purchasing workflows. It enables efficient handling of requisitions, purchase orders, vendor records, and approval processes through a structured and user-friendly interface built using Spring Boot and Thymeleaf.

---

## 🚀 Key Features
- End-to-end procurement workflow management  
- Requisition creation and tracking  
- Purchase order generation and management  
- Vendor registration and management  
- Role-based approval workflows  
- Basic reporting and record tracking  
- User-friendly web interface  

---

## ⚙️ System Architecture
1. **Requisition Management**  
   - Create and manage purchase requests  
   - Track status through approval stages  

2. **Vendor Management**  
   - Maintain vendor records and details  
   - Associate vendors with procurement activities  

3. **Purchase Order Processing**  
   - Generate and manage purchase orders  
   - Link orders with requisitions and vendors  

4. **Approval Workflow**  
   - Multi-level approval system  
   - Ensures validation and authorization of requests  

5. **Reporting Module**  
   - View and manage procurement records  
   - Basic insights into transactions  

---

## 🛠️ Technologies Used
- **Backend:** Spring Boot (Java)  
- **Frontend:** Thymeleaf, HTML, CSS  
- **Database:** MySQL  
- **Architecture:** MVC (Model-View-Controller)  
- **Tools:** Maven, Git

- Implements a structured **procurement lifecycle system**
